# Launch Audit — Lontario

Goal: ship a public job posting that accepts resume + GitHub URL applications.

## Initial audit (Wave 1) findings

Each `wave1-*.md` is written by a specialist agent. Format:

- **Severity**: `BLOCKER` (cannot ship) / `HIGH` / `MEDIUM` / `LOW`
- **Effort**: `S` (<2h) / `M` (half day) / `L` (1+ day)
- **File**: path:line
- **Finding**: what's wrong
- **Fix**: concrete remediation

Wave 2 agents tackle BLOCKER + HIGH findings. Wave 3 hardens.
